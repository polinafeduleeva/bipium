Интеграция бипиум и виалон
Раз в сутки скрипт берёт данные по машинам с сайта виалон и переносит их в каталог поездки.
Действия:
 1. с помощью токена генерируется is сессии 
 2. Получение системных id и id машин для которых будет получаться информация
 3. В цикле для каждой машины генерируется отчёт и получаются его данные
 4. Для каждой строки таблицы ищется запись с подходящей датой и машиной
 5. В каждую такую запись записываются значения для пробегов и топлива
